Scarf Command Line Arguments:

Example usage:
  scarf [options] <-raw path/to/image.raw>

-conf <path/to/conf.yaml>:
    Specifies the path to the configuration yaml file.

-csv_file <csv_file_output>:
    Specify the output CSV file.

-debug:
    Prints out additional information to debug the program.

--h || --help:
    Prints out the help information.

-log <path>:
    Specifies the log path rather than $PWD/scarf_log

-no_log:
    Will not print out the transcript

-raw <path/to/.raw or sd card>:
    Specifies the path to the raw image to pull data from. You can specify the SD card itself (ie /dev/mmc0) or a .raw file created with dd that is an image of the SD card (ie /home/user/sd_card.raw).

-quiet:
    Suppresses terminal output

-version || --version:
    Print out version info and exit.
